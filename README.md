# JavaScript Modules

## Check rules that are in conflict with Prettier
```
$ npm run eslint-check
```

## ESLint and Prettier
```
$ npm run lint
```

fix command.
```
$ npm run lint -- --fix
```

## Jest unit test
```
$ npm run test
```

watch command.
```
$ npm run test -- --watch
```

## ESDoc
```
$ npm run esdoc
```

## ESLint and ESDoc
```
$ npm run precommit
```
